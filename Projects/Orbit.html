Detumbling Algorithm


Detumbling Controller

Currently Implementating in C++ a detumbling algorithm using PID Control theory and B-dot controllers. The Proportional Integral Derivative Controller utilizes the errors in order to regulate and control the stiring of the satellite using a B-Dot Methodology. The B-Dot methodology requires to know the precise location of the satellite in order to interact with the known magnetic field at that point with the magnetorquers and set the satellite in an equilibrium or desired position.
